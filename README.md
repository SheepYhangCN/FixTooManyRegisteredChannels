# FixTooManyRegisteredChannels

Fix `Too Many Registered Channels` error when connecting to a server.<br>
Require Mod Menu & YACL<br>
Needs to open configureation interface in Mod Menu and enable it, it's disabled by default

修复连接至服务器时遇到的 `Too Many Registered Channels` 错误<br>
需要 Mod Menu 与 YACL<br>
需要在 Mod Menu 中打开配置介面并启动开关，默认不启用
